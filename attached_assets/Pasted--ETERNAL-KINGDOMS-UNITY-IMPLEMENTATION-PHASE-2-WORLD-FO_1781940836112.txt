PHASE U2.3 — WORLD STREAMING SYSTEM
===================================

Implement dynamic chunk streaming.

Create:

WorldStreamingManager

Requirements:

Only load chunks near camera.

Streaming radius:

3 chunks visible.

1 chunk preload buffer.

Unload distant chunks automatically.

Architecture must support:

2048x2048 world without performance degradation.

Use object pooling.

No Instantiate/Destroy spam.
