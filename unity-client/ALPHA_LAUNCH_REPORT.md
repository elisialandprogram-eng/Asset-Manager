### Integration Architecture
When the art team delivers an asset to the Addressables catalog:
1. `ArtImportManager` auto-detects it on next startup validation
2. `RegisterBuildingPrefab()` / `RegisterMonsterPrefab()` auto-assigns to scene controllers
3. `BuildingUpgradeVisualController` tier keys pull the correct prefab for each building level
4. `KingdomBeautificationManager` / `WorldBeautificationManager` pick up kingdom/world props
5. `AlphaVFXController` loads VFX Graph assets via pool

**Zero code changes required for art integration — 100% data-driven.**
